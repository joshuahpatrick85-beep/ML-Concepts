# N-Gram Language Models (Bigram)

Implements a bigram language model from scratch on Herman Melville's *Moby Dick* (pulled directly from Project Gutenberg): sentence-boundary tokenization, unigram/bigram counting, maximum-likelihood bigram probabilities, Laplace (add-1) smoothing, a text generator that samples from the learned distribution, and perplexity evaluation on a held-out test split.

## What's here

- Corpus loading straight from Project Gutenberg's plain-text *Moby Dick*, with manual boilerplate stripping (locates the novel's actual first and last lines to exclude the Gutenberg license header/footer)
- Sentence tokenization with `<s>`/`</s>` boundary tokens, with punctuation handling refined over several passes: comma/colon/semicolon stripping, discovering this edition uses curly quotes (`"` `"`) rather than straight quotes, and fixing a bug where sentences ending on a quoted word weren't triggering sentence closure
- `bigram_prob(w2, w1)`: maximum-likelihood bigram probability via count ratios
- `bigram_prob_smoothed`: Laplace (add-1) smoothing
- A text generator that samples from the bigram distribution starting at `<s>` until it reaches `</s>` or a length cap
- Perplexity computed on a held-out 10% test split, with counts built only from the training split

## Results

- Final vocabulary: 20,587 unique tokens; 114,768 unique bigrams
- Perplexity on held-out test set: 5018.19 (down from an initial 6285 across several tokenization cleanup passes)
