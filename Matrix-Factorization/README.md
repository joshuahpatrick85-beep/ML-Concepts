# Matrix Factorization for Movie Rating Prediction

Implementation of matrix factorization via stochastic gradient descent (SGD) to predict user movie ratings, based on the MovieLens 100K dataset — the real-world analog of the Netflix Prize-style problem covered in the course chapter.

## What's here

Ratings are loaded into a sparse `(user, movie) → rating` structure rather than a dense matrix, since the dataset only contains entries for ratings that actually exist. Two latent factor matrices, `A` (users × r) and `B` (movies × r), are randomly initialized and then iteratively updated via SGD: for each observed rating, the model predicts `A_u · B_i`, computes the error against the true rating, and nudges both `A_u` and `B_i` in the direction that reduces that error — using rank `r = 5` and learning rate `eta = 0.01` over 50 epochs. RMSE is tracked on both the training set and a held-out 80/20 test split.

## Results

- RMSE_train = 0.8126
- RMSE_test = 0.9471
