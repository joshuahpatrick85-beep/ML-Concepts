# ML Self-Study

Independent study project working through applied machine learning, built during summer 2026. Each chapter pairs a course concept with a from-scratch implementation — closed-form solutions and core algorithms are built by hand before reaching for library shortcuts, so the code reflects actual understanding of the math, not just API calls.

Practice problems for each chapter were generated by Claude from course notes (Princeton COS 324). I implemented all solutions independently and used Claude as a tutor to troubleshoot bugs and verify reasoning. See each chapter's README for specifics.
