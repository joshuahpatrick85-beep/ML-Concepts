# Reinforcement Learning: Q-Learning on Gridworld, Verified Against a Value-Iteration Baseline

## Goal
Implement Q-learning on a Gridworld environment, then check its learned policy against a value-iteration solve of the same environment, used as a ground-truth answer key.

## Approach
**Phase 1 — value iteration (known environment):** Implemented the textbook's 5×5 Gridworld (wormholes at A/B, off-grid penalty, γ=0.9). Ran value iteration for 300 sweeps to convergence, extracted the optimal value function and policy.

**Phase 2 — Q-learning (model-free):** Same Gridworld, but the agent has no upfront knowledge of rewards or transitions and learns Q(s,a) purely through experience. Used epsilon-greedy action selection (ε starting at 0.9999, decaying by a factor of 0.95 per episode down to 0.01), the standard Q-learning update rule, and episodes of 100 steps from randomized starting states. Tracked total reward per episode and plotted the learning curve. Extracted the greedy policy from the final Q-table and compared it against Phase 1's value-iteration policy by inspection.

## Results
- Value iteration converged to the values reported in the textbook's Table 14.1 (e.g., V(0,0) ≈ 21.98, V(0,1) ≈ 24.42).
- Final learned Q-values after training: Q((0,1), 'up') ≈ 1.94, Q((0,3), 'left') ≈ 18.45.
- The learned policy largely agreed with the value-iteration policy. Where they diverged, the mismatches were concentrated at the two wormhole states (A at (0,1), B at (0,3)) — action-independent states where every direction produces an identical outcome, so any disagreement there reflects a tie rather than a learning error.
