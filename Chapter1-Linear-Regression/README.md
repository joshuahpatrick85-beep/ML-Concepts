# Chapter 1: Linear Regression

Implements linear regression from scratch using the closed-form normal equations (`(XᵀX)⁻¹XᵀY`), no `sklearn` for the core solver. Built around a synthetic housing-price dataset, then extended to a text classification task.

## What's here

- Synthetic data generator (`generate_housing_data`) with a seeded RNG for reproducibility
- Manual train/test split (160/40) via array slicing, no `sklearn.model_selection`
- Closed-form solver implemented directly from the normal equations
- Train vs. test MSE comparison, plus a full-dataset refit to check how MSE shifts with more data
- A bag-of-words feature extractor applied to a small sentiment dataset, fit with linear regression and rounded to 0/1 for classification

## Results

Learned weights `[3.097, -1.465]` closely recovered the true weights `[3.0, -1.5]`. Train/test MSE landed near 1.0, consistent with the injected noise (`noise_std=1.0`) — meaning the residual error is irreducible noise, not underfitting. The bag-of-words regression classifier hit 100% accuracy on the small training set (small n, evaluated in-sample — not a claim of generalization), and highlighted the core issue: linear regression predicts a continuous value, so classification requires an arbitrary rounding threshold rather than a principled decision boundary.
