# Dimensionality Reduction, Clustering & Classification

Three algorithms implemented from scratch on the sklearn digits dataset (1797 images, 8×8 handwritten digits, 10 classes): PCA via SVD, k-means clustering, and multinomial (softmax) logistic regression. Built as one integrated project since the pipeline flows naturally from one technique into the next — reduce dimensions, cluster the result, then compare against a supervised classifier trained on the same data.

## What's here

- SVD-based PCA implemented directly from `np.linalg.svd` (center data, project onto top singular vectors) — no `sklearn.decomposition.PCA`
- 2D projection of the digits, plotted twice: colored by true label, then colored by k-means cluster assignment, for a visual before/after comparison
- k-means from scratch: random centroid initialization from real data points, iterative assignment/update loop (100 iterations, k=10)
- Cluster-to-label alignment via majority vote per cluster, producing a real accuracy score for unsupervised clustering
- A softmax (multinomial logistic) classifier trained via gradient descent from scratch — no `sklearn.linear_model` — with manual softmax, cross-entropy gradient, and train/test evaluation

## Results

- k-means clustering accuracy (post label-alignment): ~58% (varies slightly run to run due to random centroid initialization)
- Softmax classifier: 100% train accuracy, 97.2% test accuracy
