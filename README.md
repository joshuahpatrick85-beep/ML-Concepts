# ML Self-Study

Independent study project working through applied machine learning, built during summer 2026. Each concept pairs course material with a from-scratch implementation — closed-form solutions and core algorithms are built by hand before reaching for library shortcuts, so the code reflects actual understanding of the math, not just API calls.

Practice problems were generated by Claude from course notes (Princeton COS 324). I implemented all solutions independently and used Claude as a tutor to troubleshoot bugs and verify reasoning. See each project's README for specifics.

## Concepts

- [Linear Regression](./Chapter1-Linear-Regression) — closed-form regression, train/test evaluation, bag-of-words classification
- [Dimensionality Reduction, Clustering & Classification](./Dimensionality-Reduction-Clustering-Classification) — PCA via SVD, k-means clustering, softmax classification
- [N-Gram Language Models](./N-Gram-Language-Models) — bigram language model, Laplace smoothing, text generation, perplexity evaluation
- [Matrix Factorization](./Matrix-Factorization) — SGD-based matrix factorization for movie rating prediction (MovieLens 100K)
- [Feedforward Neural Network & Backpropagation](./Feedforward-Neural-Network-Backpropagation) — FFNN and backprop from scratch, verified against a textbook worked example, trained on `load_digits()`
- [Convolutional Neural Network](./Convolutional-Neural-Network) — CNN vs. FFNN baseline on CIFAR-10 using PyTorch's `nn.Conv2d`/pooling layers
- In progress — concept TBD
