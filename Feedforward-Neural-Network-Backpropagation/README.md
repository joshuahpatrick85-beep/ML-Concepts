# Feedforward Neural Network & Backpropagation (from scratch)

Implementation of forward propagation and backpropagation from scratch in numpy (no PyTorch/autograd), first verified against a textbook worked example, then applied to a real classification task on `load_digits()`.

## What's here

A toy 3→2→2→3 ReLU network is built using the exact weight matrices from the course text's worked example, and forward propagation is confirmed to reproduce the book's own numbers exactly (`h1=(0,2)`, `h2=(4,0)`, `o=(-4,8,4)`, softmax ≈ (0.00, 0.98, 0.02)) before moving on to backpropagation on the same toy network — output-layer gradients via the softmax-minus-one-hot simplification, hidden-layer error signals via the ReLU-derivative mask, and weight gradients via outer products.

That backprop logic is then rewritten as vectorized `forward()`/`backprop()` functions and applied to a real 64→32→10 FFNN (ReLU hidden layer, softmax output) trained on `load_digits()`, with an 80/20 train/test split, pixel values scaled to [0,1], and full-batch gradient descent (not mini-batch SGD — a deliberate choice given the small dataset size, 1437 training examples) over 10,000 iterations.

## Results

- Train accuracy: 0.9958
- Test accuracy: 0.9361
